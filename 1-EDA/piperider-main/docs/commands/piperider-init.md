
# piperider init
None
## Usage
```
Usage: piperider init [OPTIONS]
```
## Options
* `no_auto_search`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--no-auto-search`

  Don't search for dbt projects


* `dbt_project_dir`: 
  * Type: Path 
  * Default: `none`
  * Usage: `--dbt-project-dir`

  Directory of dbt project config


* `dbt_profiles_dir`: 
  * Type: Path 
  * Default: `none`
  * Usage: `--dbt-profiles-dir`

  Directory of dbt profiles config


* `debug`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--debug`

  Enable debug mode


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
