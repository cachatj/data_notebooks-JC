* `base`:

  * Default: `none`

* `input`: 
  * Type: Path 
  * Default: `none`
  * Usage: `--input`

  Path of the json report file to compare


* `debug`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--debug`

  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`
